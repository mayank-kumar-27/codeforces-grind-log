# Codeforces Grind Log

A structured collection of my solved Codeforces problems, serving as a personal practice log and quick reference.

---

## About

This repository contains my accepted solutions to Codeforces problems. Each problem is organized by **problem ID and name**, with solutions added as they are completed.

---

## Languages Used

- **C**
- **Python**
- **Java**

Problems may have solutions in one or multiple languages depending on the implementation approach.

---

## Repository Structure

Each folder represents a single problem:
```
Codeforces-Grind-Log/
├── 4A_Watermelon/
│   └── solution.c
│   └── solution.java
├── 41A_Translation/
│   ├── solution.c
│   └── solution.py
├── 59A_Word/
│   └── solution.py
├── 112A_Petya_and_Strings/
│   └── solution.c
└── ...
```

**Naming convention:** `<problem_id>_<problem_name>/`

All solution files follow the pattern: `solution.<ext>`
